# Desktop Text Assistant

A lightweight Python desktop assistant that allows users to quickly launch applications, open websites, and perform searches using a simple popup interface.

## Current Features
- Chat interface
- Command execution
- Basic file search

## Features to be Added
- Auto-correction for typed commands
- Flexible command parsing (word order independent)
- Smart file indexing
- WhatsApp automation
- Context memory
- Natural language task execution
- System controls (volume, brightness, screenshots)
## Features

- Open desktop applications
- Open websites in Chrome
- Google search support
- YouTube search support
- Smart URL detection
- Keyboard shortcut activation (Ctrl + Shift + Space)
- Clean Tkinter-based interface

## Technologies Used

- Python
- Tkinter
- keyboard
- subprocess
- webbrowser

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the project

```bash
python main.py
```

## Usage

Press:

```
Ctrl + Shift + Space
```

Example commands:

```
open github
open calculator
search python programming
youtube machine learning
google data science roadmap
```
